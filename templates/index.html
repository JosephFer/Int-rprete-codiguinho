Documentación de Codiguinho

  • Palabras clave de tipos de datos:

    • intnho: tipo de dato entero.
    • floatinho : tipo de dato float.
    • boolinho : tipo de dato boleano.
    • stringho : tipo de dato String.
    • verda : True.
    • falso : False.

  • Operadores matemáticos:

    • + : suma
    • - : resta
    • * : multiplicación
    • / : división
    • % : modulo

  • Operadores lógicos:

    • && : Compuerta lógica AND
    • || : Compuerta lógica OR
    • ! : NOT

  • Palabras clave de control de flujo:

    • Imprime : imprime en pantalla
    • ifisinho : sentencia condicional if.
    • elifsinho : sentencia condicional elif.
    • elsinho : sentencia condicional else.
    • darMole : sentencia ciclica while.
    • daStop : break.
    • volte : return.

  • Símbolos de control:

    • $ : define el inicio de la declaración de una variable.
    • ; : define un salto de línea.
    • = : define una asignación.
    • “ ” : delimitan cadenas de caracteres (strings).
    • {} : delimitan bloques de código.
